# Cybersecurity Thesis Lab

Repository tecnico a supporto della tesi magistrale in Sicurezza Informatica.

## Titolo della tesi

**Rilevazione e analisi di tecniche Living-off-the-Land mediante telemetria, digital forensics e supporto quantitativo**

## Obiettivo

Questo repository documenta l’ambiente sperimentale utilizzato per analizzare tecniche Living-off-the-Land, raccogliere eventi di sicurezza, produrre metriche quantitative e rappresentare graficamente i risultati ottenuti.

## Componenti principali

- Laboratorio con Wazuh, Caldera e Tsurugi
- Raccolta eventi tramite Sysmon e PowerShell Logging
- Analisi forense degli artefatti Windows
- Mapping delle tecniche su MITRE ATT&CK
- Produzione di grafici e indicatori quantitativi
- Regole Sigma, YARA e Wazuh
